# Huffman Compressor / Decompressor

Реализация алгоритма сжатия Хаффмана на C++17 с поддержкой кодирования и декодирования.

[![C++17](https://img.shields.io/badge/C++-17-blue.svg)](https://en.cppreference.com/w/cpp/17)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## Особенности

- Полноценное сжатие и восстановление файлов без потерь.
- Потоковая обработка с буферизированным чтением.
- Собственный бинарный формат с магическим числом и версионированием.
- Автоматическое освобождение памяти после работы.
- Демонстрация эволюции проекта через ветки и теги в Git.

## Как использовать

### Компиляция

```bash
g++ -std=c++17 -O2 src/huffman.cpp -o huffman
```

### Сжатие файла
```bash
./huffman -c input.txt output.huf
```
- input.txt — исходный файл (любой бинарный).
- output.huf — сжатый файл в формате Huffman.

### Распаковка
```bash
./huffman -d output.huf restored.txt
```
- output.huf — сжатый файл.
- restored.txt — восстановленный файл, идентичный исходному.
